Installation
============

Requirements
------------

- Python 3.10+
- JAX/JAXLIB
- NumPyro
- DSPS
- dustmaps

Install from source
-------------------

.. code-block:: bash

   git clone https://github.com/burke86/jaxqsofit.git
   cd jaxqsofit
   pip install -e .

Optional test dependencies
--------------------------

.. code-block:: bash

   pip install pytest pytest-cov astroquery
