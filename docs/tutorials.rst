-------------------------

- Fetching an SDSS spectrum near ``(ra, dec) = (184.0307, -2.2383)``
- Running ``QSOFit`` with ``nuts``, ``optax``, and ``optax+nuts``
- Overriding priors via ``prior_config``
- Computing broad-line FWHM and luminosity from fitted components
